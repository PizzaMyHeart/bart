# How to Use

## Installation

Installation is optional; you can use Bart as a regular website. However, you can also install Bart as an offline app on your mobile device for use without an internet connection. 

<Tabs>
    <TabItem value="android" label="Android" default>
        Supported browsers: Firefox, Chrome
    </TabItem>
    <TabItem value="ios" label="iOS">
        1. Tap the "share" button.<br/>
        2. Select "Add to Home Screen".<br/>
        3. Press the "Add" button.
    </TabItem>
</Tabs>

## Navigation

<Tabs>
    <TabItem value="mobile" label="Mobile" default>
        Click on the hamburger icon on the top left to show the index of guidelines.<br/><br/>
    </TabItem>
    <TabItem value="desktop" label="Desktop">
        The sidebar on the left shows the index of guidelines.
    </TabItem>
</Tabs>

The top of each page contains a dropdown menu with the table of contents for that page. (Hint: a scroll-to-top button appears when you start scrolling up).
